OVERVIEW

A role-based E-Commerce Management System built using ASP.NET Core Web API and SQL Server. The application manages products, orders, billing, and delivery workflows through separate user roles.

FEATURES  
Authentication & Authorization  
JWT Authentication        
Role-Based Access Control (RBAC)  
Secure API endpoints  
User Roles  
Admin  
Manage users  
Manage products and inventory    
View dashboards and reports  
Salesman  
Create customer orders  
Manage customer information  
Generate bills  
Delivery Partner  
View assigned deliveries  
Update delivery status  
Track completed deliveries  
Customer  
Browse products  
Place orders  
View order history  
Order Workflow  
Pending → Accepted → Billed → Delivered  
Dashboard  
Pending Orders Count  
Accepted Orders Count  
Billed Orders Count  
Delivered Orders Count
